AgentScope is a scope firewall and audit layer for AI coding agents. It watches what they change and blocks policy violations — before they reach git.

Get Started → See it in action ↓
$ cargo install agentscope 📋
agentscope check

Why AgentScope?

Your agent fixed the bug. It also rewrote your auth module. AgentScope tells you — before you commit.

🚫

Policy Engine

Deterministic glob-based rules. Block .env, auth files, migrations — anything sensitive. No AI judgment needed.

🧠

LLM Judge

Ask a local LLM whether the changes match your mission. Runs on Ollama — no data leaves your machine. Pick any model.

📊

Live TUI Dashboard

Watch file changes in real time with bar charts, pie charts, health bars. Press 'j' to run the judge inline.

🔒

Git Pre-commit Hook

Install with one command. Every commit runs agentscope check. Policy violations block the commit automatically.

⚡

Agent-Agnostic

Works with any agent that writes to your filesystem. Claude Code, Codex, Cursor, Gemini CLI, Copilot, and more.

🔍

Full Audit Trail

Every session is logged. Review past sessions, export JSON for CI, share markdown reports in PRs.

Every command you need

Claude Code / Codex-style CLI with model management, config, hooks, and more.

agentscope start "mission" Begin a session with a stated mission
agentscope check Check changes against mission + policy
agentscope judge -m llama3 Run LLM judge with any model
agentscope model list List available Ollama models
agentscope model set qwen3.5:2b Set default model
agentscope report Beautiful terminal report dashboard
agentscope diff --problems Quick view of blocked/unasked files
agentscope hook install Auto-check on every git commit
agentscope config show View all settings
agentscope config set provider claude Switch to Claude API
agentscope watch Live TUI with charts + inline judge
agentscope audit last-5 Review past sessions
